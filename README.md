## Features

* Multi-guild support
* Parties
* Factions
* Custom modes
* Support for multiple games

## Installation

```powershell
git clone "git@github.com/openrbw/bot.git" && cd bot
pnpm install
```

## Deploy

```powershell
pnpm run start
```
